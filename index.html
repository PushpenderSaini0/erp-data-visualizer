ERP Data Visualizer

Erp Data Visualizer is a simple tool to plot the absent summary data from ERP of ncuindia into beautiful and informative graphs.

We respect privacy, no data is stored on our server and all the processing is done client side. We take the ASP Session ID to fetch your absent summary data.
You can find the ASP Session ID in the cookies after a successful login in erp website.
You can also fetch data using your username and password, all communication are HTTPS encrypted and data is processed on your machine


Tip : Enter demo as Session ID to fetch a demo data
