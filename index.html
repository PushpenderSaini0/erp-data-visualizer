ERP Data Visualizer

Erp Data Visualizer is an open source tool to plot the absent summary data from ERP of ncu indiainto beautiful and informative graphs and tables.

We take the ASP Session ID to fetch your absent summary data.
You can find the ASP Session ID in the cookiesafter a successful login in erp website.
Data is fetched by calling a Google Cloud Function, We respect privacy, no data is logged and all the processing is done client side. You can also fetch data using your username and password, all communication are HTTPS encrypted.
You can also take a look at the source code here.


Tip : Enter demo as Session ID to fetch a demo data
